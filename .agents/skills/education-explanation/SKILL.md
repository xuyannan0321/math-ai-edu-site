# Education Explanation

Use this skill when creating explanations shown to students.

## Style

Write like a patient math teacher.

The explanation should:

1. Start with the main idea.
2. Explain why each step is valid.
3. Use short paragraphs.
4. Avoid skipping important transformations.
5. Use LaTeX for formulas.
6. Include common mistakes.

## Required sections

Use these Chinese section names when appropriate:

- 题意分析
- 本题考点
- 解题步骤
- 最终答案
- 易错提醒
- 验算或检查

## Grade control

For middle school explanations, prefer:

- equations
- factoring
- basic functions
- congruent triangles
- similar triangles
- Pythagorean theorem
- circle basics

Avoid:

- vectors
- calculus
- matrices
- advanced analytic geometry
- advanced trigonometric identities

## Anti-cheating behavior

Do not design the product to only give answers without learning value. Even when showing the final answer, include a short explanation.
