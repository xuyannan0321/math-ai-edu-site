USER GUIDE

? 帮助中心

帮助您快速了解原题真解 Pro 的使用方式、数据迁移方法和当前功能限制。

🚀
QUICK START

快速开始

  1. 进入工作台从顶部导航进入工作台，准备试题和解题要求。
  2. 提供试题信息上传试题图片或 PDF，也可以直接输入解题指令。
  3. 生成模拟解析点击生成按钮，查看教学风格的模拟解析、代码与预览效果。
  4. 保存完整页面已有完整 HTML 时，可进入源码建站并保存到原创题库或策略库。
🧭

工作台怎么用

  • 选择图片或 PDF 后，可在浏览器本地查看缩略图或文件信息。
  • 点击“清除文件”可移除当前选择，不会上传到服务器。
  • 输入教学要求后，点击按钮生成模拟解析报告。
  • 使用“代码 / 预览”切换查看生成内容。

请注意：当前工作台尚未连接真实 AI，生成内容仅为教学结构演示。

🧩

源码建站怎么用

  • 填写科目、省份、城市、类型、年份、学校和页面标题。
  • 在源码输入区粘贴完整 HTML 或 JSON 内容。
  • 点击“渲染本地预览效果”，在大尺寸窗口中检查页面。
  • 确认后发布至原创题库，或存入策略库。

预览说明：iframe 在隔离来源中允许脚本运行；请只预览自己编写或确认可信的源码，联网资源需保持网络畅通。

💡

原创题库怎么用

  • 通过关键词、类型和年份筛选本地原创题，并按时间或标题排序。
  • 使用“查看”打开大尺寸页面预览，使用“编辑”回填源码建站表单。
  • 支持删除记录、下载 HTML 和复制源码。
  • 支持导出 JSON 备份，也可以追加导入已有 JSON 数据。
📚

策略库怎么用

  • 可按科目、省市、类型、年份、日期、学校和关键词组合筛选。
  • 支持按入库时间或标题排序，也可一键重置全部筛选条件。
  • 支持查看、编辑、删除、下载 HTML 和复制源码。
  • 通过 JSON 导入、导出完成策略页备份和迁移。
🛡️
LOCAL DATA

数据备份与迁移

数据保存位置

原创题和策略页保存在当前浏览器的 localStorage 中，不会自动上传服务器。

不同来源不互通

Live Server、GitHub Pages、不同手机及不同浏览器的数据不会自动同步。

推荐迁移方式

在原环境导出 JSON，再到新环境的对应题库中导入 JSON。

定期做好备份

清理浏览器站点数据可能导致本地题库丢失，请定期保存 JSON 备份。

⚠️
CURRENT LIMITS

当前限制

  • 当前是纯静态前端项目，未连接真实 AI。
  • 未接后端、数据库和真实登录系统。
  • 积分、提现、分享和支付均为静态演示。
  • 题库数据只保存在当前浏览器，不会跨设备自动同步。
  • 当前下载格式为 HTML；真正的 Word 文档（.docx）下载留到后端阶段实现。
💬
